End-to-end serverless pipeline: Claims enter via API Gateway → Lambda Ingestor (validates & parses) → stored in S3 (AES-256) → queued in SQS with DLQ → processed by Lambda Processor (applies HIPAA rules) → stored in DynamoDB. SNS sends rejection alerts. CloudWatch monitors everything.
